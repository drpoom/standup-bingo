## Lessons Learned
1. **Subagent timeout pattern:** Complex tasks (>4 min) often timeout. Solution: spawn simpler retries with minimal specs (~30-60 lines, single responsibility)
2. **ffmpeg filter availability:** `areverb` and `anoisesrc` filters not always available. Use basic sine/white noise + volume/echo instead
3. **Direct image generation > scripts:** `scripts/generate.sh` may not exist. Use flux-schnell/flux-pro directly via illustrator skill
4. **Rate limits on flux-pro:** 6 req/min with <$5 credit. Stagger image generations or use flux-schnell for faster turnaround
5. **Build directory matters:** Always specify full path (`/home/poomk/.openclaw/workspace/elango-surfers/`) for npm commands
6. **Parallel spawning works:** 5 subagents simultaneously maximizes throughput when tasks are atomic
