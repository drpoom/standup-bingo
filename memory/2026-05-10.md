## 🎯 Key Lessons Learned

### What Worked Well
- **Parallel execution** with `--ticker` flag prevented race conditions
- **Git snapshots per phase** enabled safe rollback (5 commits total)
- **Script-first approach** minimized agent time for repetitive work
- **Scout verification gates** caught issues before they compounded

### Issues Encountered
1. **`infer_metadata.py` path bug** — relative vs absolute path resolution (fixed in Phase 1)
2. **YAML date parsing** — values like `1231-20-19` interpreted as invalid dates (fixed by quoting)
3. **MU fiscal_period: 0000** — malformed data blocking index rebuild (22 files corrected)
4. **File count mismatch** — initial audit said 567 missing frontmatter, actual was 534, then 410, then 40 (iterative improvement)

### Process Improvements
- **Always take git snapshot BEFORE cleanup** — prevents data loss
- **Delete duplicates BEFORE injecting frontmatter** — avoids wasted work
- **Validate indices AFTER rebuild** — catches broken paths early
- **Samsung placeholder** — keep empty dirs with index.md rather than removing

## 🦅 Warren Auto-Update Protocol (Added 01:10 CET)

**User Request:** "When Warren gets information on new stocks, I want him to add it to his wiki automatically."

**Implementation:** Updated `skills/warren-investing/SKILL.md` with automatic LLM-wiki update protocol.

### Automatic 5-Step Workflow

When Warren discovers new stock data (filings, presentations, transcripts):

1. **Convert** PDF/HTML → Markdown (`markitdown`)
2. **Inject** YAML frontmatter (`inject_frontmatter.py`)
3. **Rebuild** stock index (`rebuild_index.py --ticker <TICKER>`)
4. **Update** WATCHLIST.md if new ticker
5. **Git commit** with descriptive message

### Trigger Scenarios

- User request: "Warren, fetch TSM's latest 10-K"
- Daily briefing finds material news requiring filing review
- Earnings season: New 10-Q/10-K filings detected
- Manual download: User adds PDF to `stocks/<TICKER>/`

### Safety Guards

- Duplicates detected → Skip, log to `audit/duplicates.log`
- Frontmatter injection fails → Report error, skip index update
- Already has frontmatter → Skip to index rebuild only

### Report Format

```
🦅 Wiki Updated: TSM
- Added: 10-K for FY2025
- Frontmatter: ✅ Injected
- Index: ✅ Rebuilt
- Watchlist: ✅ Updated (if new ticker)
- Commit: a3f8b2c
```
